Como ler este relatório

  • O painel foi construído pelo autor. As cinco lentes compartilham o mesmo enquadramento; portanto, onde elas concordam, trate isso como uma hipótese forte, não como prova independente.
  • A confiança é pontuada de 1 a 10 com base na qualidade da evidência: estudo causal revisado por pares > dados oficiais/políticos/financeiros > pesquisa única comissionada > analogia > preprint.
  • Fato medido e interpretação são sinalizados separadamente. Uma pontuação alta significa que os dados são sólidos, não que a leitura estratégica seja certa.
